# Agrisense | Precision Farming Solution

Agrisense is a smart farming application that collects real-time soil data using NodeMCU and sensors to recommend the most suitable crops. It helps farmers monitor soil conditions and optimize irrigation and fertilizer usage for better crop yield.

## Technologies Used
- **Hardware:** NodeMCU, soil sensors (pH, moisture, temperature)
- **Backend:** Python, Flask
- **Database:** MySQL
- **Frontend:** HTML, CSS, JavaScript
- **APIs:** OpenWeather API for weather integration

## Features
- Monitor real-time soil pH, moisture, and temperature
- Get crop recommendations based on soil conditions
- Automated suggestions for water and fertilizer management
- Web dashboard to visualize soil data and crop suggestions
